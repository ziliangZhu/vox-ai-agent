# Contributing

VoxAgent welcomes public feedback, documentation suggestions, scenario ideas, and integration questions.

## Good Contributions

- Documentation corrections.
- FAQ suggestions.
- Use case descriptions.
- Integration questions that can help improve public docs.
- Feedback about website navigation or developer onboarding.

## Not Suitable For Public Issues

- Production credentials or app secrets.
- Private phone numbers or customer data.
- Account-specific organization details.
- Security-sensitive reports.
- Confidential commercial information.

Use the official contact channels in [CONTACT.md](./CONTACT.md) for private or business-sensitive communication.

## Issue Guidelines

When opening an issue, please include:

- What page or document you are referring to.
- What you expected to find.
- What was unclear or missing.
- Whether your question is about product positioning, API integration, trial access, or production preparation.

## Pull Requests

Pull requests for public documentation improvements are welcome if the repository maintainers have enabled external contribution review.

Please keep public-facing language clear, accurate, and free of private implementation details.
